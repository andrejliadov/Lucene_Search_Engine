---index---
This directory contains the index of the Cranfield Set.

---corpus---
This directory contains the Cranfield documents, the querys, a formatted set of querys.

---indexer---
This is where all of the source code for parsing, formatting, configuration, indexing and querying is located.

---results---
This is where the query scripts store the results of the querys in a correct format.

---performance---
This is where the results of the trec_eval command are stored.

---trec_eval---
This is where the source files and the trec_eval binary are stored.
